Two Coexisting Systems

This repository contains two independent trading systems that share the core Ichimoku calculation engine (core/indicator.py) and candle buffer (core/candle_buffer.py) but are otherwise fully separate.

Signal Bot (v1)

Discord notification bot. Detects 6 Ichimoku signal types across multiple pairs and timeframes. Sends rich embeds with cloud filter and cooldown management.

bot.py + config.yaml

Trading Bot (v3)

Fully automated trade execution. 4 signal modes, signal scoring, risk management, position lifecycle (break-even, trailing stop, ichimoku exits), and a live FastAPI dashboard.

trade_bot.py + strategy_config.json

Quick Start

Install & Test

bash setup_env.sh          # or: pip install -r requirements.txt
source venv/bin/activate
pytest tests/ -v           # 375 tests, no MT5 required

Signal Bot (v1)

cp .env.example .env       # add DISCORD_WEBHOOK_URL
python bot.py              # dry_run=true by default
python bot.py --dry-run    # force dry run

Trading Bot (v3)

python trade_bot.py --sim --once    # single sim cycle (test)
python trade_bot.py --sim           # continuous sim mode
python trade_bot.py                 # live (uses MT5)

Dashboard starts automatically at http://localhost:8000
